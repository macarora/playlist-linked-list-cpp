# Playlist Manager - Linked List 🎵

A C++ playlist manager built using a custom doubly linked list 
with sentinel nodes.

## Features
- Add and insert songs into the playlist
- Navigate forward and backward through songs
- Delete songs from the playlist
- Mark songs as favourites
- Filter songs by artist or favourite status
- Load songs from a file
- Interactive and file-based command modes

## Project Structure
- `main.cpp` — Main driver and command handling
- `playlist.cpp / .h` — Playlist class
- `song.cpp / .h` — Song class
- `linked_list.h / .hpp` — Templated linked list implementation
- `node.h / .hpp` — Templated node implementation
- `TopSongs.txt` — Sample song data
- `TestCommands.txt` — Sample test commands
